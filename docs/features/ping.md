## 개요

봇이 정상 작동하는지 확인하는 커맨드. 응답 지연시간(latency)을 ms 단위로 반환한다.

## 목표

- 봇 온라인 상태 및 응답 여부 빠르게 확인

## 범위 (Scope)

**포함**:
- 슬래시 커맨드 `/ping` 응답

**제외**:
- DB 조회
- 권한 제한

## 데이터 흐름

```
사용자 /ping 입력
  → interactionCreate 이벤트
  → Date.now() - interaction.createdTimestamp 로 latency 계산
  → interaction.reply(`Pong! (Xms)`)
```
